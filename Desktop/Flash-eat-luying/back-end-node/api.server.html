User

  • Method URL Description
  • POST '/api/login' Verify userName and password match and return one user.
  • POST '/api/register' Register, return one user.
  • POST '/api/profile' If already login, return session.user.
  • GET '/api/currentuser' Get current user.
  • POST '/api/users' Create a user.
  • GET '/api/users' Get all users.
  • GET '/api/users/username/:username' Get user by username.
  • GET '/api/users/:uid' Find user by userId.
  • PUT '/api/users/:uid' Update user.
  • DELETE '/api/users/:uid' Delete user.

    • Payment

  • GET '/api/payments' Find all payments.
  • GET '/api/payments/:pid' Find payment by paymentId.
  • GET '/api/payments/users/:uid' Find payments for user.
  • POST '/api/payments' Create payment.
  • PUT '/api/payments/:pid' Update payment.
  • DELETE '/api/payments/:pid' Delete payment.

    • Review

  • GET '/api/reviews' Find all reviews.
  • GET '/api/reviews/:rid' Find review by reviewId.
  • GET '/api/reviews/users/:uid' Find reviews for user.
  • GET '/api/reviews/restaurants/:rid' Find reviews for restaurant.
  • POST '/api/reviews/users/:uid/restaurants/:rid' Create review.(userId and restaurantId not included in review object.)
  • POST '/api/reviews/restaurants/:rid/users/:uid' Create review.(userId and restaurantId not included in review object.)
  • POST '/api/reviews' Create review.(userId and restaurantId included in review object.)
  • DELETE '/api/reviews/:rid' Delete review.
  • GET '/api/profile/:username' Find reviews by username.

    • Order

  • POST '/api/orders' Find all orders.
  • POST '/api/orders/:oid' Find order by orderId.
  • GET '/api/orders/users/:uid' Find order by user.
  • GET '/api/orders/restaurants/:rid' Find order by restaurant.
  • POST '/api/orders/users/:uid/restaurants/:rid' Create order.(userId and restaurantId not included in order object.)
  • POST '/api/orders/restaurants/:rid/users/:uid' Create order.(userId and restaurantId not included in order object.)
  • POST '/api/orders' Create order.(userId and restaurantId included in order object.)
  • DELETE '/api/orders/:oid' Delete order.
  • PUT '/api/orders/:oid' Update order.

    • ShoppingCart

  • GET '/api/shoppingcart/users/:uid' Find shoppingCart for user.

    • Menu

  • GET '/api/menus/restaurants/:rid' Find menu for restaurant.
  • POST '/api/menus' Create menu.(restaurantId included.)
  • DELETE '/api/menus/:mid' Delete menu.
  • PUT '/api/menus/:mid' Add item.(item json object as req.body)
  • DELETE '/api/menus/:mid/items/:iid' Delete item.
  • GET '/api/menus/:mid/items' Find items for menu.

    • Item

  • GET '/api/items/:iid' Find item by Id.
  • PUT '/api/items/:iid' Update item.


User

  • userName: String
  • firstName: String
  • lastName: String
  • password: String
  • phone: String
  • email: String
  • address: String

Payment

  • userId: String
  • paymentType: String
  • cardType: String
  • cardHolderName: String
  • expirationDate: String
  • securityCode: String

Review

  • restaurantId: String
  • userId: String
  • timestamp: String
  • review: String
  • rating: Number

Order

  • restaurantId: String
  • userId: String
  • orderOn: String
  • items: [{ type: mongoose.Schema.ObjectId, ref: 'ItemModel' }],
  • shippedOn: String
  • total: Number
  • orderStatus: {type: String, enum: ["PAID", "UNPAID", "SHIPPED", "DELIVERED", "CLOSED", "FINISHED"], default: 'UNPAID'}

ShoppingCart

  • userId: String
  • items: [{ type: mongoose.Schema.Types.ObjectId, ref: 'ItemModel' }]

Menu

  • restaurantId: String
  • items: [{ type: mongoose.Schema.Types.ObjectId, ref: 'ItemModel' }]

Item

  • itemName: String
  • price: Number
  • quantity: Number
